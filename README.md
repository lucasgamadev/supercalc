# SuperCalc

Uma coleção de calculadoras úteis para diversas finalidades.

## Estrutura do Projeto

```text
/
├── index.html
├── assets/
│   ├── css/
│   │   └── styles.css
│   ├── js/
│   │   ├── main.js
│   │   └── components/
│   │       └── ui.js
│   ├── images/
│   │   └── logo.svg
│   └── fonts/
├── components/
│   └── header.html
├── pages/
│   └── about.html
├── tailwind.config.js
├── package.json
└── README.md
```

## Calculadoras Disponíveis

- **Calculadora de Desconto**: Calcule rapidamente o valor final após aplicar um desconto.

## Tecnologias Utilizadas

- HTML5
- CSS3 & TailwindCSS
- JavaScript

## Como Executar

1. Clone este repositório
2. Abra o arquivo `index.html` em seu navegador

## Desenvolvimento

Para trabalhar no desenvolvimento do projeto:

```bash
# Instalar dependências
npm install

# Executar em modo de desenvolvimento (com watch)
npm run dev

# Gerar build para produção
npm run build
```
